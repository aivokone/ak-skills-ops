## Getting Help

**Seravo resources**:
- Documentation: https://help.seravo.com/en/
- Developer docs: https://help.seravo.com/en/collections/929963-developer-documentation
- Support: Contact via Seravo dashboard or email
- GitHub: https://github.com/Seravo/wordpress (project template issues)

**When to contact support**:
- Infrastructure issues (Nginx, PHP versions, server resources)
- DNS configuration
- SSL certificate problems
- Backup restoration (emergency)
- Account/billing questions

**When to troubleshoot yourself**:
- Plugin/theme bugs
- Database issues (wp db commands)
- Git deployment problems
- Local development setup
- wp-cli usage
- Performance optimization

This skill provides comprehensive guidance for all Seravo development scenarios. Always prioritize safety (backups), understand environment context, and leverage Seravo's custom tooling for efficient workflows.
